# Portfólio Pessoal - Atividade Prática de Web

Este projeto foi desenvolvido como parte da atividade prática da disciplina de **Desenvolvimento Web e Móveis**. O objetivo foi criar um portfólio online pessoal funcional, demonstrando competência na estruturação, estilização e interatividade de páginas web, seguindo estritamente as diretrizes de desenvolvimento "Vanilla" (sem uso de frameworks).

### 🎯 Requisitos do Projeto
Para esta entrega, os seguintes critérios foram rigorosamente seguidos:
* **Tecnologias:** HTML5, CSS3 e JavaScript puro (sem uso de Bootstrap, Tailwind, React, etc.).
* **Estrutura:** 4 seções principais (Sobre mim, Formação, Portfólio e Contato).
* **Interatividade:** Validação de formulário de contato utilizando JavaScript.
* **Hospedagem:** Site publicado via GitHub Pages.
* **Responsividade:** Design funcional adaptado para diferentes tamanhos de tela.

### 🛠 Ferramentas Utilizadas
O desenvolvimento seguiu as melhores práticas recomendadas:
* **Editor de Código:** Visual Studio Code.
* **Controle de Versão:** Git e GitHub.
* **Ambiente de Testes:** Live Server (VS Code).

### 📝 Observações Técnicas
* O código foi desenvolvido com foco na semântica do HTML5 e na organização CSS.
* O formulário de contato possui validação em JavaScript para garantir que campos obrigatórios e formato de e-mail estejam corretos antes do envio.
* Os comentários explicativos foram inseridos no código-fonte conforme solicitado na rubrica de avaliação.
